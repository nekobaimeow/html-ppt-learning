交流 vs 直流输电——两种哲学

维度交流输电 (HVAC)直流输电 (HVDC)
变压器简单，铁芯变压器直接升降需要换流站（整流+逆变）
线路成本3 相导线2 极（±），省 1/3 导线

🔑 交流输电的"临界距离"由线路充电无功决定——线路越长，电容电流越大，超过 600-800km 后送电能力急剧下降。直流没有这个问题。

PITFALL · 电晕

电晕放电——高压线路的"隐形成本"

电晕是什么
当导线表面电场强度超过空气击穿场强（约 30kV/cm），周围空气发生局部电离，产生紫蓝色辉光和滋滋声——这就是电晕放电。它不只是好看，它消耗能量。
电晕的危害

① 功率损耗：好天气 1-2kW/km，坏天气可达数十 kW/km
② 无线电干扰 (RI)：0.1-30MHz 频段噪声
③ 可听噪声 (AN)：雨天的滋滋声可超 50dB
④ 导线腐蚀：离子轰击加速氧化

抑制措施

① 分裂导线：增大等效半径，降低表面场强
② 扩径导线：空心或填充，增加外径
③ 增大导线截面：最直接但成本最高
④ 抬高导线：增大对地距离（效果有限）

爬电比距——污秽地区的核心参数

绝缘子表面沿面放电（污闪）比空气击穿更容易发生。爬电比距 = 绝缘子表面总长度 / 系统最高电压。重污区需要更多伞裙或更大爬距的绝缘子——这就是为什么沿海/工业区的绝缘子串特别长。

PRACTICE · 参数

输电线路的"身份证"——RLC 参数是怎么来的

电阻 $R$

$R = \rho \cdot l / A$
ACSR 导线 $\rho \approx 28.3\ \Omega\cdot\text{mm}^2/\text{km}$

电感 $L$（单相）

$L = \frac{\mu_0}{2\pi} \ln\frac{D_m}{r_{\text{eq}}}$
$D_m$: 相间几何均距 $r_{\text{eq}}$: 分裂导线等效半径

电容 $C$（对地）

$C = \frac{2\pi\varepsilon_0}{\ln(H/r_{\text{eq}})}$
$H$: 导线对地等效高度

波阻抗 $Z_c$

$Z_c = \sqrt{L/C} \approx 250\text{-}400\Omega$
自然功率 SIL = $U^2/Z_c$

💡 分裂导线增大 $r_{\text{eq}}$ → 电感 $L$ 减小、电容 $C$ 增大 → 波阻抗 $Z_c$ 降低 → 自然功率 SIL 增大 → 输送能力提高。这就是分裂导线的物理本质。

PRACTICE · 无功

无功补偿：电网的"隐形搬运工"

500kV 并联电抗器 · 吸收容性无功
串联补偿电容器组 · 提高线路输电能力
无功从哪里来，到哪里去

线路充电无功：长线路空载或轻载时，线路对地电容产生大量容性无功，导致末端电压升高（Ferranti 效应）。500kV 线路每百公里产生约 10-15Mvar 充电功率。
并联电抗器：直接并在母线上，吸收容性无功，抑制工频过电压——长线路必配。

PRACTICE · 动态无功

SVC 与 STATCOM——动态无功补偿的两代技术

STATCOM 静止同步补偿器 · IGBT 电压源换流器模组
对比：常规并联电抗器 · 只能吸收固定无功
SVC (静止无功补偿器)

晶闸管控制电抗器 (TCR) + 固定电容器 (FC)
通过调节晶闸管触发角连续调节感性无功
响应时间：~20-40ms
V-I 特性：电容区出力随电压下降（$Q \propto U^2$）

STATCOM (静止同步补偿器)

基于 IGBT 的电压源换流器 (VSC)
可同时发出/吸收无功
响应时间：~5ms
V-I 特性：电容区可维持额定电流（低电压不降出力）
故障期间性能远超 SVC

⚡ STATCOM 在电压跌落时仍能维持满无功出力，而 SVC 的容性出力会随 $U^2$ 衰减——这是电网故障时 STATCOM 的核心优势。

PRACTICE · 保护

继电保护——电网的"免疫系统"

继电保护屏柜 · 数字式微机保护装置
被保护的对象：500kV 变电站设备
继电保护四性

选择性：只切除故障元件，不影响非故障部分
速动性：最快速度切除故障（主保护 20-40ms）
灵敏性：对最小故障状态也能可靠动作
可靠性：该动必动（不拒动），不该动必不动（不误动）——这对矛盾贯穿所有保护设计

PRACTICE · 主保护

两种最重要的线路保护——距离与差动

距离保护（阻抗保护）

原理：$Z = U/I$，故障时测量阻抗减小
分区：I 段 80-85% 本线（瞬时），II 段 120% 本线（0.3-0.5s），III 段后备
优点：不依赖通信，天生具备方向性
局限：受过渡电阻、系统振荡影响

电流差动保护

原理：$\sum I_{\text{in}} - \sum I_{\text{out}} \approx 0$（正常）≠ 0（区内故障）
特点：绝对选择性，不受振荡影响
要求：需要通信通道（光纤/载波）传对侧电流
适用：短线路、变压器、母线、发电机

🔑 现代 500kV 线路标配双套主保护：一套光纤电流差动 + 一套距离保护，用不同原理互补——防止单一原理死区。

安全自动装置——电网不会"多米诺"的秘密

三道防线（中国电网"安全防御体系"）
防线功能措施响应时间
第一道快速切除故障，防止扩大继电保护20-40ms

PITFALL · 稳定

电网的三种稳定——功角·电压·频率

功角稳定（同步稳定）

所有发电机保持同步运行。
失稳表现：发电机间功角差持续增大 → 失步 → 解列。
临界清除时间 (CCT)：故障后必须在此时间内切除故障，否则系统将失去同步。500kV 系统 CCT 约 80-150ms。
$P = \frac{E_1 E_2}{X} \sin\delta$

电压稳定

系统维持各节点电压在合理范围。
失稳机制：无功缺额 → 电压下降 → 负荷恢复（空调·电机）→ 无功需求更大 → 电压崩溃。
PV 曲线的鼻尖点就是电压崩溃临界点。
对策：动态无功 (STATCOM) · 低压减载

⚡ 频率稳定：发电与负荷的实时平衡。频率偏差 > ±0.2Hz（大电网）即为异常。低频 → 切负荷，高频 → 切机。新能源高渗透率下，惯量下降 → 频率变化率 (RoCoF) 增大 → 对保护和控制提出新挑战。

自测
Q1: 500kV 线路电压翻倍到 1000kV，损耗降到多少？
电流减半 → 损耗 = I²R 降为原来的 1/4（25%）。精确：$1000^2/500^2 = 4$ 倍关系。
Q2: 为什么交流电缆不能做太长？
电缆对地电容大，充电电流随长度线性增长。超过临界长度后，充电电流占用全部载流量，无法输送有功——直流没有这个问题。
Q3: GIS 的绝缘介质是什么？为什么能做得那么紧凑？
SF₆（六氟化硫）气体。SF₆ 的绝缘强度是空气的 3 倍，灭弧能力是空气的 100 倍→相间距离可以大幅缩小→GIS 占地仅为 AIS 的 20-30%。
Q4: STATCOM 比 SVC 好在哪里？
STATCOM 基于 IGBT VSC，在电压跌落时仍能维持满无功出力（恒流特性）；SVC 的容性出力随 $U^2$ 衰减。故障时 STATCOM 对电压支撑能力强得多。
